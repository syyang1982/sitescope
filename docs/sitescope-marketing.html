SiteScope

AI-Powered Website Audit Platform
v0.1.3 · sitescope.vercel.app

SiteScope is an AI-driven website audit tool that analyzes any public website across five dimensions — security, frontend design, SEO, legal compliance, and cross-dimensional consistency. It produces a structured report identifying issues and risks, categorized by severity and by audience (technical vs. business), within minutes of submission.

What SiteScope Checks

DimensionScope of Analysis
SecurityHTTP security headers (HSTS, CSP, X-Frame-Options, etc.), information leakage, technology fingerprinting, SSRF exposure, form security (CSRF, CAPTCHA)
Frontend DesignMeta tags (title, description, Open Graph), WCAG 2.1 Level A accessibility, mobile responsiveness, video autoplay behavior
SEOrobots.txt configuration, sitemap.xml validity, dead links, canonical URLs, search engine indexability
Legal CompliancePrivacy policy presence and completeness, terms of service, cookie consent mechanisms, tracking script disclosure, third-party brand authorization
Cross-DimensionalContradictions between dimensions (e.g., privacy policy states no tracking while analytics scripts are detected), consistency of claims vs. implementation

Report Structure

Technical Findings

Issues of primary concern to engineering teams: security vulnerabilities, misconfigured HTTP headers, missing CSP policies, frontend code quality, SEO technical implementation, and development-mode exposure.

Business Findings

Issues of primary concern to product and business teams: legal compliance gaps, privacy policy deficiencies, cookie consent violations, unauthorized third-party assets, brand consistency issues, and user trust implications.

Each finding is assigned a severity level: Critical Requires immediate attention   Important Should be addressed soon   Minor Suggested improvement

Report Modes

ModeOutput
Brief ScanLists identified issues and associated risks. Faster generation; suitable for a quick assessment or initial review.
Full ReportIncludes all findings from the Brief Scan, plus prioritized remediation recommendations for each issue. Suitable for handing off to development or compliance teams.

How It Works

1Enter URL
2Verify Access
3AI Crawls & Analyzes
4Report Streams Live
5Download or Copy

The platform fetches the target site's homepage HTML, HTTP response headers, robots.txt, sitemap.xml, and up to five internal subpages. This data is analyzed by an AI model against the five audit dimensions. The report is streamed to the browser in real time as it is generated.

SiteScope — Platform Features & Use Cases

Platform Features

FeatureDescription
Real-Time StreamingReports generate live in the browser. Progress indicators show which dimension is currently being analyzed — no need to wait for the full process to complete.
Bilingual SupportBoth the interface and audit reports are available in English and Chinese, selectable per request.
Multiple AI ModelsBuilt-in AI models included. Alternatively, connect your own AI provider (OpenAI, Anthropic, DeepSeek, or compatible services) using your own API credentials. Credentials are never stored or logged.
Access ControlPasscode-gated access prevents unauthorized usage. Per-IP rate limiting (default: 10 audits per hour) controls costs and prevents abuse.
Audit LoggingEvery scan is logged with requesting IP, target URL, model used, and timestamp — suitable as evidence for internal compliance reviews.
Export OptionsDownload reports as Markdown files or copy to clipboard for integration into documentation, tickets, or compliance records.

Safety & Privacy

MeasureWhat It Means
Passive Scanning OnlyThe crawler reads publicly available HTML and HTTP headers via GET requests only. It does not submit forms, modify data, or attempt exploitation.
Internal Network BlockingRequests targeting localhost, private IP ranges, or cloud metadata endpoints are automatically blocked.
No Data RetentionFetched site data and user-provided API credentials exist only in memory during processing and are discarded immediately after the request completes.
Request ValidationOrigin verification prevents cross-site misuse. All inputs are validated before processing.
